## How to make pull request
- 1. Press "Git: master" in the lower right corner. You can see all git branches here. By default your branch is master (if you are not in the master branch, you need to go to it). You must create your own branch to do a homework<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.34.00.png" width="350" /><br><br>
    - 1) If you are not in the master branch, press "Git: <another branch name>" in the lower right corner<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.43.15.png" width="350" /><br><br>
    - 2) Then choose master branch<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.43.54.png" width="350" /><br><br>
    - 3) And we want to checkout master branch<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.44.07.png" width="350" /><br><br>
- 2. Choose "New Branch"<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.50.01.png" width="350" /><br><br>
- 3. Now there is a window in front of you. You can write name of your branch. In general, when you work in projects, you will need to remember the rules for writing the branch name. We also have such rules. Your branch should be called as w`<number of week>`/hw_`<your name>`_`<your surname>` where `<number of week>` is number of homework in our course `<your name>` is your name (e.g. elina or Elina), `<your surname>` is your surname (e.g. kim or Kim). After writing the name of the branch, press create<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.39.07.png" width="350" /><br><br>
- 4. If everything went well, you will see such a message in the lower left corner:<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.39.50.png" width="350" /><br><br>
- 5. You must have a github account to make a pull request. After you create account, write your username to LMS. We will invite you to the project. Accept my invitation<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.55.30.png" width="350" /><br><br>
- 6. Then accept this in github<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.55.35.png" width="350" /><br><br>
- 7. Then go back to Android Studio and press this green button at top right corner<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.52.34.png" width="350" /><br><br>
- 8. You will see this window, select all checkboxes and add commit message<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.52.47.png" width="350" /><br><br>
- 9. Click on "commit and push" button<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.52.50.png" width="350" /><br><br>
- 10. Click on "push" button<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.52.56.png" width="350" /><br><br>
- 11. Then maybe Android Studio will ask you log in to your github account <br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.53.04.png" width="350" /><br><br>
- 12. Then go to github page, refresh it, and you will see changes <br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2013.56.37.png" width="350" /><br><br>
- 13. Press on this tab <br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2014.53.36.png" width="350" /><br><br>  
- 14. Press on the branch, that you've created <br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2014.54.12.png" width="350" /><br><br>  
- 15. Press confirm button and you will see this tab, thats all, congrats! <br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/Screenshot%202020-09-18%20at%2014.58.16.png" width="350" /><br><br> 
    
## How to do a homework
- 1. Open project<br><br>
- 2. Open "TODO" tab in the lower left corner<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.11.46.png" width="350" /><br><br>
- 3. You can see all todo's here. Todo is something to help you do your homework. Here it is described in detail what needs to be written in the code. If something remains unclear or you want to test yourself, you can review the lesson<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.13.31.png" width="350" /><br><br>
- 4. In order not to search for todo in files, you can click on the desired todo and go to it.<br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.13.41.png" width="350" /><br><img src="https://github.com/erkhabibullina/Autumn-School/blob/master/img/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202020-09-18%20%D0%B2%2011.14.06.png" width="350" /><br><br>
- 5. Read the todo carefully and follow it<br><br>
- 6. Don't forget to make a pull request!
